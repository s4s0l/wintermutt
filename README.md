# wintermutt

SSH Server, that can expose some secrets from vault 
when connecting to it.

If all you've got is an ssh key and need to 'get' some
secrets from vault, this may help.

see [USAGE.md](./USAGE.md) to learn how to use it.

## Notice 

Whole project is AI generated using locally running Qwen 3.5 models,
as an experiment: how far can we go with locally running llms 
using opencode. Pretty far I'd say. For what I can tell this tool
works, and can be pretty useful.
